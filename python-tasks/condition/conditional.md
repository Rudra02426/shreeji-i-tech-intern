

### Task 1: Number Check

Take a number from the user and check:

* Positive
* Negative
* Zero

---

### Task 2: Even or Odd

Input a number and check whether it is:

* Even
* Odd

---

### Task 3: Greater Number

Take two numbers and print:

* which one is greater
* or if both are equal

---

### Task 4: Age Eligibility

Input age and check:

* Eligible to vote (age ≥ 18)
* Not eligible

---


### Task 5: Student Result

Input marks and print:

* Pass (≥ 40)
* Fail (< 40)

Add:

* Distinction (≥ 75)

---

### Task 6: Grade System

Input marks and assign grade:

| Marks  | Grade |
| ------ | ----- |
| 90–100 | A     |
| 75–89  | B     |
| 50–74  | C     |
| < 50   | Fail  |

---

### Task 7: Leap Year

Input a year and check whether it is a **leap year**.

---

### Task 8: Character Type

Input a single character and check:

* Alphabet
* Digit
* Special character

---


### Task 9: Login System

Input:

* username
* password

Check:

* If both are correct → Login Successful
* Else → Invalid Credentials

---

### Task 10: Electricity Bill

Input units consumed and calculate bill:

| Units     | Price    |
| --------- | -------- |
| First 100 | ₹5/unit  |
| Next 100  | ₹7/unit  |
| Above 200 | ₹10/unit |

---

### Task 11: ATM Withdrawal

Input:

* account balance
* withdraw amount

Conditions:

* Minimum balance should be ₹500
* Withdraw amount should be multiple of 100
* Amount should not exceed balance

---


### Task 12: Largest of Three Numbers

Take three numbers and find the largest using:

* if-elif-else (no `max()`)

---

### Task 13: Number Range Checker

Check if a number lies between:

* 10 and 50
* 51 and 100
* Above 100

---

### Task 14: Simple Calculator

Input:

* two numbers
* operator (+, -, *, /)

Perform operation using conditionals.

---

### Task 15: Traffic Light System

Input color:

* Red → Stop
* Yellow → Ready
* Green → Go
